# 变更日志

所有重要变更都会记录在本文件中。

## [Unreleased]

### Added

- 工资管理支持按工资条分类录入：发放项（基本工资、绩效工资、租房/伙食/交通补助、加班费、加班补助、奖金）、社保/公积金缴费基数、扣除项（医疗、养老、失业、公积金）、所得税与税后附加项（大病医疗、采暖补贴），并派生应发工资、扣除项合计、税前/税后工资与实发金额。
- 支持按年额外录入年终奖（`month=0`），在统计中单独汇总为「年终奖合计」，并新增发放项构成与扣除项构成统计。

### Changed

- 工资管理启动时会检测旧版 `salary_records.allowance` / `deduction` 结构并直接重建为最新分类工资表，避免旧 `NOT NULL` 约束继续阻塞工资或年终奖录入。
- 工资录入弹窗将月份字段文案调整为“月份（年终奖可留空）”，明确勾选年终奖后由前端自动提交 `month=0`。
- 初版发布
